# Baby Monitor

A baby monitor written in Python that is controlled via telegram. 


## Dependencies

```
sudo pip3 install -r requirements.txt
sudo apt install libatlas-base-dev libjasper-dev libqtgui4 libqt4-test python3-pyqt5 python3-opencv
```
